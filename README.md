# Cinema-Demand-Forecasting – Kaggle Competition

This project was developed as part of a Kaggle competition focused on predicting cinema demand using historical movie and attendance data. The goal was to build a predictive model capable of forecasting audience demand based on past performance and related features.

The dataset was explored and cleaned using Python, followed by exploratory data analysis to understand patterns in movie performance and audience behavior. Predictive modeling techniques were applied to estimate future cinema demand.

**Key Steps**

Data cleaning and preprocessing

Exploratory Data Analysis (EDA)

Feature exploration and selection

Demand forecasting using machine learning models

Evaluation of model performance

**Tools Used**

Python

Pandas

NumPy

Matplotlib / Seaborn
Scikit-learn
